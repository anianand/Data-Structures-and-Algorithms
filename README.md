# Data-Structures-and-Algorithms
My implementation of several Data Structures and Algorithms in C++. This repo contains the following implementations.

* [**Binary Search Tree**](https://en.wikipedia.org/wiki/Binary_search_tree)
* [**Merge and Quick Sort**](http://www.cs.wcupa.edu/rkline/ds/fast-sorts.html)
* [**Convex Hull Problem**](https://en.wikipedia.org/wiki/Convex_hull)
* [**Heap Sort**](https://en.wikipedia.org/wiki/Heapsort)
* [**Strassen's Algo of Matrix Multiplication**](https://en.wikipedia.org/wiki/Strassen_algorithm)
* [**Kruskal's and Prim's Algo for finding Minimum spanning Tree in a Graph**](https://www.quora.com/What-is-the-difference-in-Kruskals-and-Prims-algorithm)
* [**Huffman Coding Problem**](www.geeksforgeeks.org/greedy-algorithms-set-3-huffman-coding)
* [**0/1 Knapsack Problem**](www.geeksforgeeks.org/dynamic-programming-set-10-0-1-knapsack-problem/)
